## ✨ Key Features

<div align="center">
  
| Feature | Description |
|---------|-------------|
| **🎥 HD Video Calls** | Crystal-clear video quality with adaptive resolution |
| **🔇 Smart Controls** | Mute audio/video with single-click toggles |
| **📞 Call Management** | Start/end calls with intuitive UI controls |
| **👥 Multi-Participant** | Supports multiple concurrent users |
| **🛠️ Admin Controls** | Host privileges for session management |
| **🚀 Low Latency** | Optimized WebRTC implementation |

## 🚀 Getting Started

### Prerequisites
- Node.js v16+
- npm/yarn
- Firebase account (for authentication)

### Installation
```bash
# Clone repository
git clone https://github.com/Raj72620/video-conference-app.git

# Install dependencies
cd video-conference-app
npm install

# Start development server
npm run dev
